# Hl7.Cql.Invocation

A support package for HL7.Cql that contains functionality to invoke CQL from libraries.

## Overview

This package provides high-level APIs for invoking CQL libraries and expressions. It simplifies the process of loading, compiling, and executing CQL libraries in .NET applications.

## Key Features

- **Library Loading**: Load and manage CQL libraries from various sources
- **Expression Invocation**: Execute specific CQL expressions and functions
- **Context Management**: Handle execution context and parameter passing
- **Result Processing**: Process and return CQL execution results
- **Integration Pipeline**: Combines CQL compilation and packaging functionality

## Usage

This package provides the main entry point for applications that need to execute CQL logic programmatically.

```csharp
// High-level APIs for invoking CQL libraries
// and executing CQL expressions in applications
```

## Dependencies

- **CodeGeneration.NET**: .NET code generation utilities
- **Hl7.Cql.CqlToElm**: CQL to ELM conversion
- **Hl7.Cql.Packaging**: CQL packaging functionality
