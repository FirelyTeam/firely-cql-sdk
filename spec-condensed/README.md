# CQL Specification - Condensed for AI Parsing

This directory contains the CQL 1.5.3 Release 1 Errata 2 specification converted to Markdown format.

## Purpose

The original specification includes 1,752 files (40MB) with CSS, JavaScript, images, and other web assets.
This condensed version contains only the specification content in Markdown format for easier AI parsing.

## Conversion Details

- **Original:** 40MB total (1,752 files including 1,000+ CSS/JS/images)
- **HTML content:** 2,055,180 bytes (23 main HTML files)
- **Condensed:** 873,655 bytes (23 Markdown files)
- **Content reduction:** 57.5% (removed navigation, styling, scripts)
- **Total size reduction:** 97.7% (40MB → 912KB)
- **Generated:** condense_spec.py
